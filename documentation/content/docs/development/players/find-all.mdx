## Find all players

Use this method to retrieve a list of all currently online players in the system.

### Synchronous

<Tabs items={['Kotlin', 'Java']}>
  <Tab value="Kotlin">
    <DynamicCodeBlock
      lang="kotlin"
      code={`val players = playerProvider.findAll()

for (player in players) {
    println("Found player: \${player.name} on \${player.currentServiceName}")
}`}
    />
  </Tab>
  <Tab value="Java">
    <DynamicCodeBlock
      lang="java"
      code={`List<PolocloudPlayer> players = playerProvider.findAll();

for (PolocloudPlayer player : players) {
    System.out.println("Found player: " + player.getName() + " on " + player.getCurrentServiceName());
}`}
    />
  </Tab>
</Tabs>

<Callout type="info">
  The list may be empty if no players are currently online.
</Callout>

### Asynchronous

Use the async version if you want to avoid blocking the thread.

<Tabs items={['Kotlin', 'Java']}>
  <Tab value="Kotlin">
    <DynamicCodeBlock
      lang="kotlin"
      code={`playerProvider.findAllAsync().thenAccept { players ->
    if (players.isEmpty()) return@thenAccept

    players.forEach {
        println("Found player: \${it.name} on \${it.currentServiceName}")
    }
}`}
    />
  </Tab>
  <Tab value="Java">
    <DynamicCodeBlock
      lang="java"
      code={`playerProvider.findAllAsync().thenAccept(players -> {
    if (players.isEmpty()) return;

    for (PolocloudPlayer player : players) {
        System.out.println("Found player: " + player.getName() + " on " + player.getCurrentServiceName());
    }
});`}
    />
  </Tab>
</Tabs>

<Callout type="tip">
  For large networks prefer <code>findAllAsync()</code> to avoid blocking the main thread.
</Callout>

## Return value

- **Sync:** <code>List&lt;PolocloudPlayer&gt;</code>  
- **Async:** <code>CompletableFuture&lt;List&lt;PolocloudPlayer&gt;&gt;</code>

All player objects returned represent players that are **currently online** and connected to services.  
Offline or disconnected players are not included.
