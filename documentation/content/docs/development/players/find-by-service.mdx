## Find players by service

Use this method to retrieve all players that are currently connected to a specific service.  
You can filter players either by passing a `Service` object or the service's name as a string.

<Callout type="tip">
  For details on the <code>Service</code> object and service management, see the <a href="../services">Services</a> reference page.
</Callout>

### Synchronous by Service object

<Tabs items={['Kotlin', 'Java']}>
  <Tab value="Kotlin">
    <DynamicCodeBlock lang="kotlin" code={`val service: Service = ... // obtain or create service instance
val players = playerProvider.findByService(service.name())

players.forEach { player ->
    println("Player: \${player.name} - Service: \${player.currentServiceName}")
}`} />
  </Tab>
  <Tab value="Java">
    <DynamicCodeBlock lang="java" code={`Service service = ...; // obtain or create service instance
List<PolocloudPlayer> players = playerProvider.findByService(service.name());

for (PolocloudPlayer player : players) {
    System.out.println("Player: " + player.name + " - Service: " + player.currentServiceName);
}`} />
  </Tab>
</Tabs>

### Synchronous by service name

<Tabs items={['Kotlin', 'Java']}>
  <Tab value="Kotlin">
    <DynamicCodeBlock lang="kotlin" code={`val players = playerProvider.findByService("lobby-1")

players.forEach { player ->
    println("Player: \${player.name} - Service: \${player.currentServiceName}")
}`} />
  </Tab>
  <Tab value="Java">
    <DynamicCodeBlock lang="java" code={`List<PolocloudPlayer> players = playerProvider.findByService("lobby-1");

for (PolocloudPlayer player : players) {
    System.out.println("Player: " + player.name + " - Service: " + player.currentServiceName);
}`} />
  </Tab>
</Tabs>

<Callout type="info">
  The returned list may be empty if no players are currently connected to the specified service.
</Callout>

## Return value

- **Sync:** <code>List&lt;PolocloudPlayer&gt;</code>  
- **Async:** <code>CompletableFuture&lt;List&lt;PolocloudPlayer&gt;&gt;</code>

The returned list contains all players currently connected to the specified service.  
Players that are offline, disconnected, or connected to other services are not included.
