## Player Provider

The `PlayerProvider` is your main entry point for all player-related operations in PoloCloud. It provides methods to find, track, and manage players across your cloud infrastructure.

## Finding Players

Discover and track players across your services:

## Player Properties

<Callout type="info">
**Player Information**

Each player object contains essential information about their current state and location in your cloud infrastructure.
</Callout>

| Property           | Type     | Description                                    |
| ------------------ | -------- | ---------------------------------------------- |
| `name`             | `String` | The player's display name                      |
| `uniqueId`         | `UUID`   | The player's unique identifier                 |
| `currentServiceName` | `String` | The service the player is currently connected to |

## Player Tracking Examples

### Monitor Player Distribution

<Tabs items={['Kotlin', 'Java']}>
    <Tab value="Kotlin">
        <DynamicCodeBlock lang="kotlin" code={`// Get player distribution across services
val allPlayers = playerProvider.findAll()
val playersByService = allPlayers.groupBy { it.currentServiceName }

playersByService.forEach { (serviceName, players) ->
    println("\$serviceName: \${players.size} players")
    players.forEach { player ->
        println("  - \${player.name} (\${player.uniqueId()})")
    }
}`} />
    </Tab>
    <Tab value="Java">
        <DynamicCodeBlock lang="java" code={`// Get player distribution across services
List<PolocloudPlayer> allPlayers = playerProvider.findAll();
Map<String, List<PolocloudPlayer>> playersByService = allPlayers.stream()
    .collect(Collectors.groupingBy(PolocloudPlayer::getCurrentServiceName));

playersByService.forEach((serviceName, players) -> {
    System.out.println(serviceName + ": " + players.size() + " players");
    players.forEach(player -> 
        System.out.println("  - " + player.getName() + " (" + player.uniqueId() + ")")
    );
});`} />
    </Tab>
</Tabs>

### Find Players by Service Type

<Tabs items={['Kotlin', 'Java']}>
    <Tab value="Kotlin">
        <DynamicCodeBlock lang="kotlin" code={`// Find all server services
val serverServices = serviceProvider.findByType(GroupType.SERVER)

// Get players from all server services
val serverPlayers = serverServices.flatMap { service ->
    playerProvider.findByService(service.name())
}

println("Total players on servers: \${serverPlayers.size}")`} />
    </Tab>
    <Tab value="Java">
        <DynamicCodeBlock lang="java" code={`// Find all server services
List<Service> serverServices = serviceProvider.findByType(GroupType.SERVER);

// Get players from all server services
List<PolocloudPlayer> serverPlayers = serverServices.stream()
    .flatMap(service -> playerProvider.findByService(service.name()).stream())
    .collect(Collectors.toList());

System.out.println("Total players on servers: " + serverPlayers.size());`} />
    </Tab>
</Tabs>

## Real-time Player Monitoring

<Callout type="tip">
**Event-driven Player Tracking**

For real-time player monitoring, consider using PoloCloud's event system to listen for player join/leave events and update your player tracking accordingly.
</Callout>

The PlayerProvider gives you powerful tools to track and manage players across your entire cloud infrastructure, making it easy to implement features like player balancing, monitoring, and analytics.
