## Overview of Group Types

In PoloCloud, services are categorized by their **GroupType**, which defines their role or classification in the system. Understanding these types helps you filter and manage services efficiently.

## GroupType Enum

The `GroupType` enum defines three main categories for services:

<DynamicCodeBlock
  lang="java"
  code={`enum GroupType {
    SERVICE,
    SERVER,
    PROXY
}`}
/>

### Filtering by Group Type

<Tabs items={['Kotlin', 'Java']}>
    <Tab value="Kotlin">
        <DynamicCodeBlock lang="kotlin" code={`// Find all server groups
val serverGroups = groupProvider.findAll()
    .filter { it.type == GroupType.SERVER }

// Find all proxy groups
val proxyGroups = groupProvider.findAll()
    .filter { it.type == GroupType.PROXY }

// Find all service groups
val serviceGroups = groupProvider.findAll()
    .filter { it.type == GroupType.SERVICE }`} />
    </Tab>
    <Tab value="Java">
        <DynamicCodeBlock lang="java" code={`// Find all server groups
List<Group> serverGroups = groupProvider.findAll().stream()
    .filter(group -> group.getType() == GroupType.SERVER)
    .collect(Collectors.toList());

// Find all proxy groups
List<Group> proxyGroups = groupProvider.findAll().stream()
    .filter(group -> group.getType() == GroupType.PROXY)
    .collect(Collectors.toList());

// Find all service groups
List<Group> serviceGroups = groupProvider.findAll().stream()
    .filter(group -> group.getType() == GroupType.SERVICE)
    .collect(Collectors.toList());`} />
    </Tab>
