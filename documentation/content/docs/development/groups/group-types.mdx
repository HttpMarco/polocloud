## Overview of Group Types

In PoloCloud, services are categorized by their **GroupType**, which defines their role or classification in the system. Understanding these types helps you filter and manage services efficiently.

## GroupType Enum

The `GroupType` enum defines three main categories for services:

<DynamicCodeBlock
  lang="java"
  code={`enum GroupType {
    SERVICE,
    SERVER,
    PROXY
}`}
/>


## Usage Examples

<Callout type="info">
**Group Type Filtering**

Use group types to filter and manage services based on their function. This helps organize your cloud infrastructure and apply type-specific configurations.

### Filtering Services by Group Type

Note: `GroupType` is used to filter **Services**, not Groups. Groups don't have a type field.

<Tabs items={['Kotlin', 'Java']}>
    <Tab value="Kotlin">
        <DynamicCodeBlock lang="kotlin" code={`// Find all server services
val serverServices = serviceProvider.findByType(GroupType.SERVER)

// Find all proxy services
val proxyServices = serviceProvider.findByType(GroupType.PROXY)

// Find all service-type services
val serviceTypeServices = serviceProvider.findByType(GroupType.SERVICE)`} />
    </Tab>
    <Tab value="Java">
        <DynamicCodeBlock lang="java" code={`// Find all server services
