## What are Groups?

In PoloCloud, a **Group** represents a logical collection or category of services. Groups help you organize and manage services by their function, environment, or any custom classification.

<div className="text-center mb-12">
    <div className="w-20 h-1 bg-gradient-to-r from-transparent via-primary/50 to-transparent rounded-full mx-auto mb-6" />
    <div className="text-xl text-muted-foreground max-w-3xl mx-auto">
        Organize your services logically for better management and scalability
    </div>
</div>


### Benefits of Using Groups

- **Logical Organization** - Categorize services by function or environment
- **Bulk Operations** - Manage multiple services simultaneously
- **Easy Discovery** - Filter and find services quickly
- **Scalability** - Set automatic scaling rules per group
- **Configuration** - Apply group-specific settings and templates

## Working with Groups

### Getting the Group Provider

First, get access to the group provider using the PoloCloud SDK:

<Tabs items={['Kotlin', 'Java']}>
    <Tab value="Kotlin">
        <DynamicCodeBlock lang="kotlin" code='val groupProvider = PoloCloud.groupProvider()' />
    </Tab>
    <Tab value="Java">
        <DynamicCodeBlock lang="java" code='var groupProvider = PoloCloud.groupProvider();' />
    </Tab>
</Tabs>

### Finding Groups

Discover existing groups in your cloud infrastructure:

<Tabs items={['Kotlin', 'Java']}>
    <Tab value="Kotlin">
        <DynamicCodeBlock lang="kotlin" code={`// Find all groups
val allGroups = groupProvider.findAll()

// Find a specific group by name
val lobbyGroup = groupProvider.find("lobby")

// Async operations
val allGroupsAsync = groupProvider.findAllAsync()
val lobbyGroupAsync = groupProvider.findAsync("lobby")`} />
    </Tab>
    <Tab value="Java">
        <DynamicCodeBlock lang="java" code={`// Find all groups
List<Group> allGroups = groupProvider.findAll();

// Find a specific group by name
Group lobbyGroup = groupProvider.find("lobby");

// Async operations
CompletableFuture<List<Group>> allGroupsAsync = groupProvider.findAllAsync();
CompletableFuture<Group> lobbyGroupAsync = groupProvider.findAsync("lobby");`} />
    </Tab>
</Tabs>

### Creating Groups

Create new groups with custom configurations:

<Tabs items={['Kotlin', 'Java']}>
    <Tab value="Kotlin">
        <DynamicCodeBlock lang="kotlin" code={`// Create a new group
val newGroup = Group(
    name = "lobby",
    minMemory = 1024,
    maxMemory = 2048,
    minOnlineService = 1,
    maxOnlineService = 5,
    platform = PlatformIndex("paper", "1.21.1"),
    percentageToStartNewService = 80.0,
    createdAt = System.currentTimeMillis(),
    templates = listOf(template),
    properties = PropertyHolder.empty()
)

// Save the group
val createdGroup = groupProvider.create(newGroup)

// Async creation
val createdGroupAsync = groupProvider.createAsync(newGroup)`} />
    </Tab>
    <Tab value="Java">
        <DynamicCodeBlock lang="java" code={`// Create a new group
Group newGroup = new Group(
    "lobby",
    1024,  // minMemory
    2048,  // maxMemory
    1,     // minOnlineService
    5,     // maxOnlineService
    new PlatformIndex("paper", "1.21.1"),
    80.0,  // percentageToStartNewService
    System.currentTimeMillis(),
    List.of(template),
    PropertyHolder.empty()
);

// Save the group
Group createdGroup = groupProvider.create(newGroup);

// Async creation
CompletableFuture<Group> createdGroupAsync = groupProvider.createAsync(newGroup);`} />
    </Tab>
</Tabs>

### Updating Groups

Modify existing group configurations by creating a new Group instance:

<Tabs items={['Kotlin', 'Java']}>
    <Tab value="Kotlin">
        <DynamicCodeBlock lang="kotlin" code={`// Create updated group with new settings
val updatedGroup = Group(
    name = lobbyGroup.name,
    minMemory = 2048,  // Updated memory
    maxMemory = 4096,  // Updated memory
    minOnlineService = lobbyGroup.minOnlineService,
    maxOnlineService = lobbyGroup.maxOnlineService,
    platform = lobbyGroup.platform,
    percentageToStartNewService = lobbyGroup.percentageToStartNewService,
    createdAt = lobbyGroup.createdAt,
    templates = lobbyGroup.templates,
    properties = lobbyGroup.properties
)

// Save changes
val savedGroup = groupProvider.update(updatedGroup)

// Async update
val savedGroupAsync = groupProvider.updateAsync(updatedGroup)`} />
    </Tab>
    <Tab value="Java">
        <DynamicCodeBlock lang="java" code={`// Create updated group with new settings
Group updatedGroup = new Group(
    lobbyGroup.getName(),
    2048,  // Updated minMemory
    4096,  // Updated maxMemory
    lobbyGroup.getMinOnlineService(),
    lobbyGroup.getMaxOnlineService(),
    lobbyGroup.getPlatform(),
    lobbyGroup.getPercentageToStartNewService(),
    lobbyGroup.getCreatedAt(),
    lobbyGroup.getTemplates(),
    lobbyGroup.getProperties()
);

// Save changes
Group savedGroup = groupProvider.update(updatedGroup);

// Async update
CompletableFuture<Group> savedGroupAsync = groupProvider.updateAsync(updatedGroup);`} />
    </Tab>
</Tabs>
