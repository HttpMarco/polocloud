## Find group by name

Use this method to find a specific group by its name.

<DynamicCodeBlock
  lang="rust"
  code={`use polocloud_sdk::PoloCloudSDK;

#[tokio::main]
async fn main() -> Result<(), Box<dyn std::error::Error>> {
    let sdk = PoloCloudSDK::new().await?;
    let mut group_provider = sdk.group_provider().lock().await?;
    
    let group_name = "lobby".to_string();
    match group_provider.find_by_name_async(group_name).await? {
        Some(group) => {
            println!("Found group: {}", group.name());
            println!("Min memory: {}MB", group.minimum_memory());
            println!("Max memory: {}MB", group.maximum_memory());
        }
        None => println!("Group not found"),
    }
    
    Ok(())
}`}
/>

<Callout type="info">
  Returns <code>None</code> if no group with the specified name exists.
</Callout>

## Return value

- **Type:** <code>Result&lt;Option&lt;Group&gt;, Status&gt;</code>

The returned group object contains all configuration information including memory limits, online service counts, and custom properties.
