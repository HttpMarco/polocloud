## Find all groups

Use this method to retrieve a list of all currently registered groups in the system.

<DynamicCodeBlock
  lang="rust"
  code={`use polocloud_sdk::PoloCloudSDK;

#[tokio::main]
async fn main() -> Result<(), Box<dyn std::error::Error>> {
    let sdk = PoloCloudSDK::new().await?;
    let mut group_provider = sdk.group_provider().lock().await?;
    
    let groups = group_provider.find_all_async().await?;
    
    for group in groups {
        println!("Found group: {}", group.name());
    }
    
    Ok(())
}`}
/>

<Callout type="info">
  The list may be empty if no groups are currently registered.
</Callout>

## Return value

- **Type:** <code>Result&lt;Vec&lt;Group&gt;, Status&gt;</code>

All group objects returned represent the **current state** of registered groups.  
Groups contain configuration information like memory limits, online service counts, and custom properties.
