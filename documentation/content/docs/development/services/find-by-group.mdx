## Find services by group

Use this method to retrieve all services that belong to a specific group.  
You can filter services either by passing a `Group` object or the group's name as a string.

<Callout type="tip">
  For details on the <code>Group</code> object and group management, see the <a href="./groups">Groups</a> reference page.
</Callout>

### Synchronous by Group object

<Tabs items={['Kotlin', 'Java']}>
  <Tab value="Kotlin">
    <DynamicCodeBlock lang="kotlin" code={`val group: Group = ... // obtain or create group instance
val services = serviceProvider.findByGroup(group)

services.forEach { service ->
    println("Service: \${service.name()} - Group: \${service.groupName}")
}`} />
  </Tab>
  <Tab value="Java">
    <DynamicCodeBlock lang="java" code={`Group group = ...; // obtain or create group instance
List<Service> services = serviceProvider.findByGroup(group);

for (Service service : services) {
    System.out.println("Service: " + service.name() + " - Group: " + service.groupName);
}`} />
  </Tab>
</Tabs>

### Synchronous by group name

<Tabs items={['Kotlin', 'Java']}>
  <Tab value="Kotlin">
    <DynamicCodeBlock lang="kotlin" code={`val services = serviceProvider.findByGroup("lobby")

services.forEach { service ->
    println("Service: \${service.name()} - Group: \${service.groupName}")
}`} />
  </Tab>
  <Tab value="Java">
    <DynamicCodeBlock lang="java" code={`List<Service> services = serviceProvider.findByGroup("lobby");

for (Service service : services) {
    System.out.println("Service: " + service.name() + " - Group: " + service.groupName);

  </Tab>

<Callout type="info">
  The returned list may be empty if no services exist in the specified group.
</Callout>

## Return value

- **Sync:** <code>List&lt;Service&gt;</code>  
- **Async:** <code>CompletableFuture&lt;List&lt;Service&gt;&gt;</code>

The returned list contains all services currently active and registered in the specified group.  
Services offline, unregistered, or provisioning are not included.
