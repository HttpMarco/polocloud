## Find all services

Use this method to retrieve a list of all currently registered or running services in the system.

### Synchronous

<Tabs items={['Kotlin', 'Java']}>
  <Tab value="Kotlin">
    <DynamicCodeBlock
      lang="kotlin"
      code={`val services = serviceProvider.findAll()

for (service in services) {
    println("Found service: \${service.name()}")
}`}
    />
  </Tab>
  <Tab value="Java">
    <DynamicCodeBlock
      lang="java"
      code={`List<Service> services = serviceProvider.findAll();

for (Service service : services) {
    System.out.println("Found service: " + service.name());
}`}
    />
  </Tab>
</Tabs>

<Callout type="info">
  The list may be empty if no services are currently active.
</Callout>

### Asynchronous

Use the async version if you want to avoid blocking the thread.

<Tabs items={['Kotlin', 'Java']}>
  <Tab value="Kotlin">
    <DynamicCodeBlock
      lang="kotlin"
      code={`serviceProvider.findAllAsync().thenAccept { services ->
    if (services.isEmpty()) return@thenAccept

    services.forEach {
        println("Found service: \${it.name()}")
    }
}`}
    />
  </Tab>
  <Tab value="Java">
    <DynamicCodeBlock
      lang="java"
      code={`serviceProvider.findAllAsync().thenAccept(services -> {
    if (services.isEmpty()) return;

    for (Service service : services) {
        System.out.println("Found service: " + service.name());
    }
});`}
    />
  </Tab>
</Tabs>

<Callout type="tip">
  For large networks prefer <code>findAllAsync()</code> to avoid blocking the main thread.
</Callout>

## Return value

- **Sync:** <code>List&lt;Service&gt;</code>  
- **Async:** <code>CompletableFuture&lt;List&lt;Service&gt;&gt;</code>

All service objects returned represent the **current runtime state** of active services.  
Offline, unregistered, or booting services are not included.
