## Service Management

Services in PoloCloud represent individual Minecraft server instances running in your cloud infrastructure. The SDK provides powerful tools to discover, manage, and interact with these services.

</div>

## Initialization

Get access to the service provider using the PoloCloud SDK:

<Tabs items={['Kotlin', 'Java']}>
    <Tab value="Kotlin">
        <DynamicCodeBlock lang="kotlin" code='val serviceProvider = Polocloud.instance().serviceProvider()' />
    </Tab>
    <Tab value="Java">
        <DynamicCodeBlock lang="java" code='SharedServiceProvider&lt;Service&gt; serviceProvider = Polocloud.instance().serviceProvider();' />
    </Tab>
</Tabs>

## Starting Services

Boot new service instances with custom configurations:

<Tabs items={['Kotlin', 'Java']}>
    <Tab value="Kotlin">
        <DynamicCodeBlock lang="kotlin" code={`// Start a service with default group settings
val serviceSnapshot = serviceProvider.bootInstance("lobby")

// Start a service with custom configuration
val bootConfig = SharedBootConfiguration()
    .withMinMemory(2048)
    .withMaxMemory(4096)
    .withTemplate(lobbyTemplate)
    .withProperty("custom-property", "value")
val customServiceSnapshot = serviceProvider.bootInstanceWithConfiguration("lobby", bootConfig)`} />
    </Tab>
    <Tab value="Java">
        <DynamicCodeBlock lang="java" code={`// Start a service with default group settings
ServiceSnapshot serviceSnapshot = serviceProvider.bootInstance("lobby");

// Start a service with custom configuration
SharedBootConfiguration bootConfig = new SharedBootConfiguration()
    .withMinMemory(2048)
    .withMaxMemory(4096)
    .withTemplate(lobbyTemplate)
    .withProperty("custom-property", "value");
ServiceSnapshot customServiceSnapshot = serviceProvider.bootInstanceWithConfiguration("lobby", bootConfig);`} />
    </Tab>
</Tabs>

## Stopping Services

Shutdown running service instances:

<Tabs items={['Kotlin', 'Java']}>
    <Tab value="Kotlin">
        <DynamicCodeBlock lang="kotlin" code={`// Shutdown a service by name
val shutdownSnapshot = serviceProvider.shutdownService("lobby-1")

// Shutdown using service instance
val service = serviceProvider.find("lobby-1")
service?.shutdown()`} />
    </Tab>
    <Tab value="Java">
        <DynamicCodeBlock lang="java" code={`// Shutdown a service by name
ServiceSnapshot shutdownSnapshot = serviceProvider.shutdownService("lobby-1");

// Shutdown using service instance
Service service = serviceProvider.find("lobby-1");
if (service != null) {
    service.shutdown();
}`} />
    </Tab>
</Tabs>

## Service Properties

<Callout type="info">
**Service Information**

Each service contains detailed information about its current state, performance metrics, and configuration.
</Callout>

| Property        | Type           | Description                                    | Access Method |
| --------------- | -------------- | ---------------------------------------------- | ------------- |
| `groupName`     | `String`       | The group this service belongs to              | Direct field |
| `id`            | `Int`          | Unique service identifier within the group     | Direct field |
| `name()`        | `String`       | Full service name (groupName-id)              | Method |
| `state`         | `ServiceState` | Current state (STARTING, ONLINE, STOPPING, STOPPED) | Direct field |
| `type`          | `GroupType`    | Service type (SERVICE, SERVER, PROXY)          | Direct field |
| `hostname`      | `String`       | Network hostname for connections               | Direct field |
| `port`          | `Int`          | Network port for connections                   | Direct field |
| `playerCount`   | `Int`          | Current number of connected players            | Direct field |
| `maxPlayerCount`| `Int`          | Maximum number of players allowed              | Direct field |
| `memoryUsage`   | `Double`       | Current memory usage percentage                | Direct field |
| `cpuUsage`      | `Double`       | Current CPU usage percentage                   | Direct field |
| `motd`          | `String`       | Message of the day displayed to players        | Direct field |
| `properties`    | `Map<String, String>` | Custom service properties                 | Direct field |
| `templates`     | `List<Template>` | Templates used for this service            | Direct field |
| `minMemory`     | `Int`          | Minimum memory allocation                      | Direct field |
| `maxMemory`     | `Int`          | Maximum memory allocation                      | Direct field |
| `information`   | `ServiceInformation` | Additional service information          | Direct field |
