## Template Provider

The `TemplateProvider` gives you access to manage and work with templates in your PoloCloud infrastructure. Templates define the configuration and files that are used when creating new services.

## Finding Templates

Discover and retrieve templates from your cloud infrastructure:

<Tabs items={['Kotlin', 'Java']}>
    <Tab value="Kotlin">
        <DynamicCodeBlock lang="kotlin" code={`// Find all templates
val allTemplates = templateProvider.findAll()

val lobbyTemplate = templateProvider.find("lobby-template")

// Async operations
val allTemplatesAsync = templateProvider.findAllAsync()
val lobbyTemplateAsync = templateProvider.findAsync("lobby-template")`} />

## Template Properties

<Callout type="info">
**Template Information**

Each template contains essential information about its configuration, files, and usage in your cloud infrastructure.
</Callout>

| Property        | Type           | Description                                    |
| --------------- | -------------- | ---------------------------------------------- |
| `name`          | `String`       | The template's unique identifier               |
| `size`          | `Long`         | Total size of all files in the template        |
| `files`         | `List<File>`   | List of files included in this template        |
| `properties`    | `Map<String, String>` | Template-specific configuration properties |

## Template Management Examples

### List All Templates

<Tabs items={['Kotlin', 'Java']}>
    <Tab value="Kotlin">
        <DynamicCodeBlock lang="kotlin" code={`// Get all templates and display information
val templates = templateProvider.findAll()

templates.forEach { template ->
    println("Template: \${template.name}")
    println("  Size: \${template.size()} bytes")
    println("  Files: \${template.files.size}")
    println("  Properties: \${template.properties}")
    println("---")
}`} />
    </Tab>
    <Tab value="Java">
        <DynamicCodeBlock lang="java" code={`// Get all templates and display information
List<Template> templates = templateProvider.findAll();

for (Template template : templates) {
    System.out.println("Template: " + template.getName());
    System.out.println("  Size: " + template.size() + " bytes");
    System.out.println("  Files: " + template.getFiles().size());
    System.out.println("  Properties: " + template.getProperties());
    System.out.println("---");
}`} />
    </Tab>
</Tabs>

### Async Template Operations

<Tabs items={['Kotlin', 'Java']}>
    <Tab value="Kotlin">
        <DynamicCodeBlock lang="kotlin" code={`// Async template operations
templateProvider.findAllAsync().thenAccept { templates ->
    println("Found \${templates.size} templates")
    templates.forEach { template ->
        println("- \${template.name} (\${template.size()} bytes)")
    }
}

templateProvider.findAsync("lobby-template").thenAccept { template ->
    if (template != null) {
        println("Template found: \${template.name}")
    } else {
        println("Template not found")
    }
}`} />
    </Tab>
    <Tab value="Java">
        <DynamicCodeBlock lang="java" code={`// Async template operations
templateProvider.findAllAsync().thenAccept(templates -> {
    System.out.println("Found " + templates.size() + " templates");
    for (Template template : templates) {
        System.out.println("- " + template.getName() + " (" + template.size() + " bytes)");
    }
});

templateProvider.findAsync("lobby-template").thenAccept(template -> {
    if (template != null) {
        System.out.println("Template found: " + template.getName());
    } else {
        System.out.println("Template not found");
    }
});`} />
    </Tab>
</Tabs>
