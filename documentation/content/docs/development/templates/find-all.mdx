## Find all templates

Use this method to retrieve a list of all available templates in your cloud infrastructure.

### Synchronous

<Tabs items={['Kotlin', 'Java']}>
  <Tab value="Kotlin">
    <DynamicCodeBlock
      lang="kotlin"
      code={`val templates = templateProvider.findAll()

for (template in templates) {
    println("Found template: \${template.name} (\${template.size()})")
}`}
    />
  </Tab>
  <Tab value="Java">
    <DynamicCodeBlock
      lang="java"
      code={`List<Template> templates = templateProvider.findAll();

for (Template template : templates) {
    System.out.println("Found template: " + template.name + " (" + template.size() + ")");
}`}
    />
  </Tab>
</Tabs>

<Callout type="info">
  The list may be empty if no templates are currently available.
</Callout>

### Asynchronous

Use the async version if you want to avoid blocking the thread.

<Tabs items={['Kotlin', 'Java']}>
  <Tab value="Kotlin">
    <DynamicCodeBlock
      lang="kotlin"
      code={`templateProvider.findAllAsync().thenAccept { templates ->
    if (templates.isEmpty()) return@thenAccept

    templates.forEach {
        println("Found template: \${it.name} (\${it.size()})")
    }
}`}
    />
  </Tab>
  <Tab value="Java">
    <DynamicCodeBlock
      lang="java"
      code={`templateProvider.findAllAsync().thenAccept(templates -> {
    if (templates.isEmpty()) return;

    for (Template template : templates) {
        System.out.println("Found template: " + template.name + " (" + template.size() + ")");
    }
});`}
    />
  </Tab>
</Tabs>

<Callout type="tip">
  For large template collections prefer <code>findAllAsync()</code> to avoid blocking the main thread.
</Callout>

## Return value

- **Sync:** <code>List&lt;Template&gt;</code>  
- **Async:** <code>CompletableFuture&lt;List&lt;Template&gt;&gt;</code>

All template objects returned represent templates that are **currently available** in your cloud infrastructure.  
Templates that are being processed, corrupted, or temporarily unavailable are not included.
