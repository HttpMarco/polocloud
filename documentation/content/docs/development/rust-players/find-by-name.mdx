## Find player by name

Use this method to find a specific player by their name.

<DynamicCodeBlock
  lang="rust"
  code={`use polocloud_sdk::PoloCloudSDK;

#[tokio::main]
async fn main() -> Result<(), Box<dyn std::error::Error>> {
    let sdk = PoloCloudSDK::new().await?;
    let mut player_provider = sdk.player_provider().lock().await?;
    
    let player_name = "Player123".to_string();
    match player_provider.find_by_name_async(player_name).await? {
        Some(player) => {
            println!("Found player: {}", player.name());
            println!("UUID: {}", player.uuid());
            println!("Current service: {}", player.curren_service_name());
        }
        None => println!("Player not found"),
    }
    
    Ok(())
}`}
/>

<Callout type="info">
  Returns <code>None</code> if no player with the specified name is currently connected.
</Callout>

## Return value

- **Type:** <code>Result&lt;Option&lt;PolocloudPlayer&gt;, Status&gt;</code>

The returned player object contains all current information including name, UUID, and current service.
