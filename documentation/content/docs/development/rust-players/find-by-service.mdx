## Find players by service

Use this method to find all players currently connected to a specific service.

<DynamicCodeBlock
  lang="rust"
  code={`use polocloud_sdk::PoloCloudSDK;

#[tokio::main]
async fn main() -> Result<(), Box<dyn std::error::Error>> {
    let sdk = PoloCloudSDK::new().await?;
    let mut player_provider = sdk.player_provider().lock().await?;
    
    let service_name = "lobby-1".to_string();
    let players = player_provider.find_by_service_async(service_name).await?;
    
    for player in players {
        println!("Found player: {} (UUID: {})", 
                player.name(), 
                player.uuid());
    }
    
    Ok(())
}`}
/>

<Callout type="info">
  Returns an empty vector if no players are currently connected to the specified service.
</Callout>

## Return value

- **Type:** <code>Result&lt;Vec&lt;PolocloudPlayer&gt;, Status&gt;</code>

All returned players are currently connected to the specified service.
