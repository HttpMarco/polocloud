## Find all players

Use this method to retrieve a list of all currently connected players in the system.

<DynamicCodeBlock
  lang="rust"
  code={`use polocloud_sdk::PoloCloudSDK;

#[tokio::main]
async fn main() -> Result<(), Box<dyn std::error::Error>> {
    let sdk = PoloCloudSDK::new().await?;
    let mut player_provider = sdk.player_provider().lock().await?;
    
    let players = player_provider.find_all_async().await?;
    
    for player in players {
        println!("Found player: {} (UUID: {})", 
                player.name(), 
                player.uuid());
    }
    
    Ok(())
}`}
/>

<Callout type="info">
  The list may be empty if no players are currently connected.
</Callout>

## Return value

- **Type:** <code>Result&lt;Vec&lt;PolocloudPlayer&gt;, Status&gt;</code>

All player objects returned represent the **current state** of connected players.  
Offline or disconnected players are not included.
