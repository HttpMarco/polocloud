## Find services by name

Use this method to find services by their name.

<DynamicCodeBlock
  lang="rust"
  code={`use polocloud_sdk::PoloCloudSDK;

#[tokio::main]
async fn main() -> Result<(), Box<dyn std::error::Error>> {
    let sdk = PoloCloudSDK::new().await?;
    let mut service_provider = sdk.service_provider().lock().await?;
    
    let service_name = "lobby-1".to_string();
    let services = service_provider.find_by_name_async(service_name).await?;
    
    for service in services {
        println!("Found service: {} ({}:{})", 
                service.group_name, 
                service.hostname, 
                service.port);
    }
    
    Ok(())
}`}
/>

<Callout type="info">
  Returns an empty vector if no services with the specified name exist.
</Callout>

## Return value

- **Type:** <code>Result&lt;Vec&lt;Service&gt;, Status&gt;</code>

The returned service objects contain all runtime information including hostname, port, and current state.
