## Find services by group

Use this method to find all services belonging to a specific group object.

<DynamicCodeBlock
  lang="rust"
  code={`use polocloud_sdk::PoloCloudSDK;

#[tokio::main]
async fn main() -> Result<(), Box<dyn std::error::Error>> {
    let sdk = PoloCloudSDK::new().await?;
    let mut service_provider = sdk.service_provider().lock().await?;
    let mut group_provider = sdk.group_provider().lock().await?;
    
    // First get the group
    let group_name = "lobby".to_string();
    if let Some(group) = group_provider.find_by_name_async(group_name).await? {
        // Find services by group object
        let services = service_provider.find_group_async(group).await?;
        
        for service in services {
            println!("Found service: {} ({}:{})", 
                    service.group_name, 
                    service.hostname, 
                    service.port);
        }
    }
    
    Ok(())
}`}
/>

<Callout type="info">
  Returns an empty vector if no services exist in the specified group.
</Callout>

## Return value

- **Type:** <code>Result&lt;Vec&lt;Service&gt;, Status&gt;</code>

All returned services belong to the specified group and contain their runtime information.
