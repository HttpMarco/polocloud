## Find all services

Use this method to retrieve a list of all currently registered or running services in the system.

<DynamicCodeBlock
  lang="rust"
  code={`use polocloud_sdk::PoloCloudSDK;

#[tokio::main]
async fn main() -> Result<(), Box<dyn std::error::Error>> {
    let sdk = PoloCloudSDK::new().await?;
    let mut service_provider = sdk.service_provider().lock().await?;
    
    let services = service_provider.find_all_async().await?;
    
    for service in services {
        println!("Found service: {} ({}:{})", 
                service.group_name, 
                service.hostname, 
                service.port);
    }
    
    Ok(())
}`}
/>

<Callout type="info">
  The list may be empty if no services are currently active.
</Callout>

## Return value

- **Type:** <code>Result&lt;Vec&lt;Service&gt;, Status&gt;</code>

All service objects returned represent the **current runtime state** of active services.  
Offline, unregistered, or booting services are not included.
