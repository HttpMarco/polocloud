## Send events

Use the `call` method to send events to other parts of your cloud infrastructure.

### Basic Usage

<Tabs items={['Kotlin', 'Java']}>
  <Tab value="Kotlin">
    <DynamicCodeBlock
      lang="kotlin"
      code={`// Send a custom event
val customEvent = MyCustomEvent("Hello from service!")
eventProvider.call(customEvent)

// Send built-in events
val playerJoinEvent = PlayerJoinEvent(player)
eventProvider.call(playerJoinEvent)

val serviceChangeEvent = ServiceChangeStateEvent(service)
eventProvider.call(serviceChangeEvent)`}
    />
  </Tab>
  <Tab value="Java">
    <DynamicCodeBlock
      lang="java"
      code={`// Send a custom event
MyCustomEvent customEvent = new MyCustomEvent("Hello from service!");
eventProvider.call(customEvent);

// Send built-in events
PlayerJoinEvent playerJoinEvent = new PlayerJoinEvent(player);
eventProvider.call(playerJoinEvent);

ServiceChangeStateEvent serviceChangeEvent = new ServiceChangeStateEvent(service);
eventProvider.call(serviceChangeEvent);`}
    />
  </Tab>
</Tabs>

<Callout type="info">
  Events are sent asynchronously. The <code>call</code> method returns immediately and the event is processed in the background.
</Callout>

## Custom Events

Create and send your own custom events:

<Tabs items={['Kotlin', 'Java']}>
  <Tab value="Kotlin">
    <DynamicCodeBlock
      lang="kotlin"
      code={`// Define a custom event
data class MyCustomEvent(val message: String) : Event

// Send the custom event
val event = MyCustomEvent("Custom message from service")
eventProvider.call(event)

// Send with additional data
data class ServiceMetricsEvent(
    val serviceName: String,
    val cpuUsage: Double,
    val memoryUsage: Double
) : Event

val metricsEvent = ServiceMetricsEvent("lobby-1", 45.2, 67.8)
eventProvider.call(metricsEvent)`}
    />
  </Tab>
  <Tab value="Java">
    <DynamicCodeBlock
      lang="java"
      code={`// Define a custom event
public class MyCustomEvent implements Event {
    private final String message;
    
    public MyCustomEvent(String message) {
        this.message = message;
    }
    
    public String getMessage() {
        return message;
    }
}

// Send the custom event
MyCustomEvent event = new MyCustomEvent("Custom message from service");
eventProvider.call(event);

// Send with additional data
public class ServiceMetricsEvent implements Event {
    private final String serviceName;
    private final double cpuUsage;
    private final double memoryUsage;
    
    public ServiceMetricsEvent(String serviceName, double cpuUsage, double memoryUsage) {
        this.serviceName = serviceName;
        this.cpuUsage = cpuUsage;
        this.memoryUsage = memoryUsage;
    }
    
    // Getters...
}

ServiceMetricsEvent metricsEvent = new ServiceMetricsEvent("lobby-1", 45.2, 67.8);
eventProvider.call(metricsEvent);`}
    />
  </Tab>
</Tabs>

## Built-in Events

Send common system events:

  <Tab value="Kotlin">

eventProvider.call(serviceChangeEvent)

ServiceChangeStateEvent serviceChangeEvent = new ServiceChangeStateEvent(service);

<Callout type="warning">
  Make sure your custom events implement the <code>Event</code> interface and are properly serializable for transmission across the network.
</Callout>

## Return value

- **Type:** <code>void</code>
- **Behavior:** Asynchronous - method returns immediately

The event is queued for delivery and will be processed by any subscribers in the background.
