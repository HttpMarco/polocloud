## Find a platform by name

Once you've initialized the `PlatformProvider`, you can look up a specific platform by its name:

### Synchronous

<Tabs items={['Kotlin', 'Java']}>
  <Tab value="Kotlin">
    <DynamicCodeBlock lang="kotlin" code='val platform = platformProvider.find("paper")' />
  </Tab>
  <Tab value="Java">
    <DynamicCodeBlock lang="java" code='Platform platform = platformProvider.find("paper");' />
  </Tab>
</Tabs>

<Callout type="warning">
  This method returns <code>null</code> if no platform with the given name is found.
</Callout>

## Find platforms by type

You can also find all platforms of a specific type:

### Synchronous

<Tabs items={['Kotlin', 'Java']}>
  <Tab value="Kotlin">
    <DynamicCodeBlock lang="kotlin" code={`// Find all server platforms
val serverPlatforms = platformProvider.find(GroupType.SERVER)

// Find all proxy platforms
val proxyPlatforms = platformProvider.find(GroupType.PROXY)

// Find all service platforms
val servicePlatforms = platformProvider.find(GroupType.SERVICE)`} />
  </Tab>
  <Tab value="Java">
    <DynamicCodeBlock lang="java" code={`// Find all server platforms
List<Platform> serverPlatforms = platformProvider.find(GroupType.SERVER);

// Find all proxy platforms
List<Platform> proxyPlatforms = platformProvider.find(GroupType.PROXY);

// Find all service platforms
List<Platform> servicePlatforms = platformProvider.find(GroupType.SERVICE);`} />
  </Tab>
</Tabs>

<Callout type="info">
  The type-based search returns a <code>List&lt;Platform&gt;</code> which may be empty if no platforms of the specified type are found.
</Callout>

## Return value

- **By name:** <code>Platform</code> or <code>null</code>  
- **By type:** <code>List&lt;Platform&gt;</code>

The returned `Platform` object represents the **current state** of the requested platform.  
If no platform with the specified name exists or is available, the result is `null`.
