## Find all platforms

Use this method to retrieve a list of all available platforms in your cloud infrastructure.

### Synchronous

<Tabs items={['Kotlin', 'Java']}>
  <Tab value="Kotlin">
    <DynamicCodeBlock
      lang="kotlin"
      code={`val platforms = platformProvider.findAll()

for (platform in platforms) {
    println("Found platform: \${platform.name} \${platform.version} (\${platform.type})")
}`}
    />
  </Tab>
  <Tab value="Java">
    <DynamicCodeBlock
      lang="java"
      code={`List<Platform> platforms = platformProvider.findAll();

for (Platform platform : platforms) {
    System.out.println("Found platform: " + platform.getName() + " " + platform.getVersion() + " (" + platform.getType() + ")");
}`}
    />
  </Tab>
</Tabs>

<Callout type="info">
  The list may be empty if no platforms are currently available.
</Callout>

## Return value

- **Type:** <code>List&lt;Platform&gt;</code>

All platform objects returned represent platforms that are **currently available** in your cloud infrastructure.  
Platforms that are being processed, corrupted, or temporarily unavailable are not included.
