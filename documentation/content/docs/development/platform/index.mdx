## Platform Provider

The `PlatformProvider` gives you access to manage and work with platforms in your PoloCloud infrastructure. Platforms define the server software and versions that can be used for creating services.

### Initialization

<Tabs items={['Kotlin', 'Java']}>
    <Tab value="Kotlin">
        <DynamicCodeBlock lang="kotlin" code='val platformProvider = PoloCloud.platformProvider()' />
    </Tab>
    <Tab value="Java">
        <DynamicCodeBlock lang="java" code='var platformProvider = PoloCloud.platformProvider();' />
    </Tab>
</Tabs>

## Finding Platforms

Discover and retrieve platforms from your cloud infrastructure:

<Tabs items={['Kotlin', 'Java']}>
    <Tab value="Kotlin">
        <DynamicCodeBlock lang="kotlin" code={`// Find all platforms
val allPlatforms = platformProvider.findAll()

// Find a specific platform by name
val paperPlatform = platformProvider.find("paper")

// Find platforms by type
val serverPlatforms = platformProvider.find(GroupType.SERVER)
val proxyPlatforms = platformProvider.find(GroupType.PROXY)`} />
    </Tab>
    <Tab value="Java">
        <DynamicCodeBlock lang="java" code={`// Find all platforms

// Find a specific platform by name
Platform paperPlatform = platformProvider.find("paper");

// Find platforms by type
List<Platform> serverPlatforms = platformProvider.find(GroupType.SERVER);
List<Platform> proxyPlatforms = platformProvider.find(GroupType.PROXY);`} />
    </Tab>
</Tabs>

## Platform Properties

<Callout type="info">
**Platform Information**

Each platform contains essential information about the server software, version, and compatibility with different group types.
</Callout>

| Property        | Type           | Description                                    |
| --------------- | -------------- | ---------------------------------------------- |
| `name`          | `String`       | The platform's unique identifier               |
| `version`       | `String`       | Version of the platform                        |
| `type`          | `GroupType`    | Type of platform (SERVER, PROXY, SERVICE)      |
| `description`   | `String`       | Human-readable description of the platform     |
| `properties`    | `Map<String, String>` | Platform-specific configuration properties |

## Platform Management Examples

### List All Platforms

<Tabs items={['Kotlin', 'Java']}>
    <Tab value="Kotlin">
        <DynamicCodeBlock lang="kotlin" code={`// Get all platforms and display information
val platforms = platformProvider.findAll()

platforms.forEach { platform ->
    println("Platform: \${platform.name}")
    println("  Version: \${platform.version}")
    println("  Type: \${platform.type}")
    println("  Description: \${platform.description}")
    println("---")
}`} />
    </Tab>
    <Tab value="Java">
        <DynamicCodeBlock lang="java" code={`// Get all platforms and display information
List<Platform> platforms = platformProvider.findAll();

for (Platform platform : platforms) {
    System.out.println("Platform: " + platform.getName());
    System.out.println("  Version: " + platform.getVersion());
    System.out.println("  Type: " + platform.getType());
    System.out.println("  Description: " + platform.getDescription());
    System.out.println("---");
}`} />
    </Tab>
</Tabs>

### Find Platform by Name

<Tabs items={['Kotlin', 'Java']}>
    <Tab value="Kotlin">
        <DynamicCodeBlock lang="kotlin" code={`// Find a specific platform
val platform = platformProvider.find("paper")

if (platform != null) {
    println("Found platform: \${platform.name} \${platform.version}")
    println("Type: \${platform.type}")
} else {
    println("Platform not found")
}`} />
    </Tab>
    <Tab value="Java">
        <DynamicCodeBlock lang="java" code={`// Find a specific platform
Platform platform = platformProvider.find("paper");

if (platform != null) {
    System.out.println("Found platform: " + platform.getName() + " " + platform.getVersion());
    System.out.println("Type: " + platform.getType());
} else {
    System.out.println("Platform not found");
}`} />
    </Tab>
</Tabs>
