## Supported Platforms

The Proxy Addon supports multiple proxy platforms:

- **Velocity** - High-performance proxy for modern Minecraft networks
- **BungeeCord** - Popular proxy solution with extensive plugin support

## Available Commands

The Proxy Addon provides a comprehensive command system accessible via `/polocloud` (or aliases `/cloud`, `/p`):

### Service Management
- **`/polocloud list`** - List all available services with player counts and status
- **`/polocloud start <groupName>`** - Start a new service from the specified group
- **`/polocloud stop <service>`** - Stop a specific service
- **`/polocloud info`** - Display general PoloCloud information

### Player Management
- **`/polocloud players`** - Show all online players and their current servers
- **`/polocloud broadcast <message>`** - Send a message to all online players

### Maintenance Control
- **`/polocloud maintenance <on|off> [group]`** - Enable/disable maintenance mode for all services or specific group

## Configuration Options

The Proxy Addon creates a `proxy-config.json` file with extensive customization options:

### Basic Settings
- **`prefix`** - Command prefix for all messages
- **`aliases`** - Command aliases (default: `["cloud", "p"]`)

### Message Customization
All messages are fully customizable with placeholders:
- **`no_permission`** - Permission denied message
- **`only_players`** - Console-only command message
- **`starting`** - Service start confirmation
- **`stopping`** - Service stop confirmation
- **`no_players_online`** - No players online message
- **`no_server_found`** - No services found message

### MOTD Configuration
- **`motd.enabled`** - Enable/disable MOTD updates
- **`motd.lineOne`** - First line of server list MOTD
- **`motd.lineTwo`** - Second line of server list MOTD
- **`motd.pingMessage`** - Ping message for server list

### Maintenance MOTD
- **`maintenanceMotd.enabled`** - Enable maintenance mode MOTD
- **`maintenanceMotd.lineOne`** - Maintenance mode first line
- **`maintenanceMotd.lineTwo`** - Maintenance mode second line
- **`maintenanceMotd.pingMessage`** - Maintenance mode ping message

### Tablist Configuration
- **`tablist.enabled`** - Enable/disable tablist updates
- **`tablist.header`** - Tablist header text
- **`tablist.footer`** - Tablist footer text

## Installation

1. **Download** the Proxy Addon from the PoloCloud releases
2. **Place** the addon JAR in your `/polocloud/addons/` directory
3. **Restart** PoloCloud to load the addon
4. **Configure** proxy settings in your group configuration

## Getting Started

1. **Install the Addon** by placing the JAR file in `/polocloud/addons/`
2. **Restart PoloCloud** to load the addon
3. **Configure Messages** in the generated `proxy-config.json` file
4. **Set up MOTD and Tablist** with your custom branding
5. **Test Commands** using `/polocloud` in-game or console

## Message Placeholders

The Proxy Addon supports various placeholders in messages:

- **`%service%`** - Service name
- **`%group%`** - Group name  
- **`%player%`** - Player name
- **`%server%`** - Server name
- **`%players%`** - Current player count
- **`%maxPlayers%`** - Maximum player count
- **`%count%`** - Total player count
- **`%serverCount%`** - Number of servers
