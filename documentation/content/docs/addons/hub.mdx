## Overview

The **Hub Addon** provides fallback server management and hub commands for PoloCloud. It allows players to easily connect to designated fallback servers (hubs/lobbies) and provides a centralized way to manage server navigation in your network.

## Supported Platforms

The Hub Addon supports multiple proxy platforms:

- **Velocity** - High-performance proxy with MiniMessage support
- **BungeeCord** - Popular proxy solution with legacy text formatting

## How It Works

The Hub Addon works by:

1. **Scanning for Fallback Servers** - Searches for services with `fallback: true` property
2. **Registering Commands** - Registers `/hub` command with configurable aliases
3. **Handling Connections** - Manages player connections to fallback servers
4. **Providing Feedback** - Sends appropriate messages based on connection status

## Configuration

### Group Configuration

To use the Hub Addon, you need to mark at least one server as a fallback server in your group configuration:

```json
{
  "name": "lobby-group",
  "type": "SERVER",
  "platform": "paper",
  "properties": {
    "fallback": "true"
  }
}
```

### Addon Configuration

The Hub Addon creates a `hub-config.json` file with the following options:

- **`prefix`** - Command prefix for all messages
- **`aliases`** - Command aliases (default: `["lobby", "l"]`)
- **`messages`** - Customizable command messages

## Available Commands

The Hub Addon provides the following command:

### Hub Command
- **Command**: `/hub` (aliases: `/lobby`, `/l`)
- **Description**: Connects the player to the designated fallback server
- **Permission**: None (available to all players)
- **Usage**: `/hub`

## Configuration Example

Here's a complete example of the `hub-config.json` file:

```json
{
  "prefix": "§8[§bPoloCloud§8] ",
  "messages": {
    "no_permission": "§cYou have no permission!",
    "only_players": "§cOnly players can use this command!",
    "no_fallback_found": "§cNo fallback server found!",
    "connected_to_fallback": "§aYou have been connected to the fallback server §e{server}§a!",
    "already_connected_to_fallback": "§cYou are already connected to the fallback server §e{server}§c!"
  },
  "aliases": ["lobby", "l"]
}
```

## Message Placeholders

The Hub Addon supports the following placeholders in messages:

- **`{server}`** - Fallback server name

## Installation

1. **Download** the Hub Addon from the PoloCloud releases
2. **Place** the addon JAR in your `/polocloud/addons/` directory
3. **Restart** PoloCloud to load the addon
4. **Configure** a fallback server in your group settings
5. **Test** the `/hub` command

## Getting Started

1. **Install the Addon** by placing the JAR file in `/polocloud/addons/`
2. **Restart PoloCloud** to load the addon
3. **Configure Fallback Server** by setting `fallback: true` in your group properties
4. **Test Commands** using `/hub` in-game
5. **Customize Messages** in the generated `hub-config.json` file
