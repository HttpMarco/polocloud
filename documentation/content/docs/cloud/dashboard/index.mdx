## Dashboard Sections

### Service Status
- **Live Status** - Real-time service status indicators
- **Player Counts** - Current players on each service
- **Resource Usage** - CPU, memory, and network utilization
- **Uptime Tracking** - Service availability and performance metrics

### Network Overview
- **Group Management** - View and manage all service groups
- **Platform Status** - Monitor different platform types (Paper, Velocity, etc.)
- **Template Management** - Manage service templates and configurations
- **Load Distribution** - Visual representation of player distribution

### System Information
- **Server Performance** - CPU, memory, and disk usage
- **Network Health** - Connection status and latency monitoring
- **Event Logs** - Real-time system events and notifications
- **Configuration Status** - Current settings and recent changes
