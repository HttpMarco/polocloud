## Overview

The **Hosted Dashboard** is our managed dashboard service that provides easy setup, automatic updates, and maintenance-free operation for your PoloCloud infrastructure.

## Prerequisites

Before using the hosted dashboard, ensure you have:

- **PoloCloud Core** - Version 3.0.0 or higher with REST Module
- **Network Access** - Dashboard needs to connect to your PoloCloud instance

## Installation Steps

### Step 1: Install PoloCloud

First, you need to install PoloCloud Core. Follow our comprehensive installation guide:

[**→ Install PoloCloud**](/docs/cloud/installation)

### Step 2: Install REST Module

Install the REST Module to enable API access for the dashboard:

[**→ Install REST Module**](/docs/modules/rest-module/installation)

### Step 3: Access Dashboard Onboarding

Navigate to the hosted dashboard onboarding page:

[**→ Dashboard Onboarding**](https://dashboard.polocloud.de/onboarding)

### Step 4: Configure Connection

Enter your PoloCloud server details:

- **IP Address** - Your PoloCloud server IP
- **Port** - The port you configured for the REST Module (default: 8080)

### Step 5: Create Admin Account

Set up your admin account with the following details:

- **Minecraft Username** - Your Minecraft player name
- **Password** - Secure password for dashboard access
- **Confirm Password** - Re-enter your password

### Step 6: Access Dashboard

Your hosted dashboard is now ready to use!

## Benefits of Hosted Dashboard

- **Easy Setup** - Get started in minutes without server configuration
- **Automatic Updates** - Always have the latest features and security patches
- **No Maintenance** - We handle all server maintenance and updates
- **High Availability** - Reliable service with 99.9% uptime
